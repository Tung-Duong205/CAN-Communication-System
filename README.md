# CAN Communication System

This project implements a CAN Bus communication system using ESP32 and SN65HVD230 CAN Transceiver Modules. 
## System Overview
The system consists of three nodes:
- Engine Node
- Fuel Node
- Dashboard Node

These nodes communicate through the CAN Bus network to exchange vehicle-related data in real time.

## Files
- Node_engine_26_5_2024.ino
- Node_fuel_26_5_2026.ino
- node_tablo_final_30_5_2026.ino

## Hardware
- ESP32
- SN65HVD230 CAN Transceiver Module 
- CAN Transceiver
- LCD/OLED Display (Dashboard Node)
- Encoder V2
- CJMCU-75
- A yellow DC gear motor
- PWM DC motor
- A potentiometer
- 5V and 3.3V power supplies
